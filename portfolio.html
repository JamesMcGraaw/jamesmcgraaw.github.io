Group Project - Music Player API

The aim of this project was to make an API to connect a pre-existing front end with a provided database. We had been studing OOP and this project was designed and executed to follow SOLID principles. The API needed to output the data requested at a number of different endpoints in JSON format. I worked on the architecture of our file structure. I also worked on the /artists endpoint to take all of the artists, their albums and the songs on each album from the database to be displayed on the page.
